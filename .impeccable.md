## Design Context

### Users
**Main app**: Gen-Z / young adults seeking genuine AI companionship — emotionally vulnerable, craving belonging, texting from their phones in casual moments. They want to feel like they walked into a real friend group that's always online.

**Admin dashboard**: Single founder/operator. Opening it under pressure (something broke, costs spiking, user issue). Needs to read the situation fast and act decisively. Zero tolerance for confusion.

### Brand Personality
**Three words: Cozy · Intimate · Human**

MyGang is a safe space that feels like texting your actual best friends — warm, unpretentious, real. Casual and Gen-Z-coded ("gang," "vibe") but the visual execution is polished. Think luxury clubhouse with a sense of humor. Earns trust through familiarity, not flash.

### Aesthetic Direction

**Main app:**
- Dark-first glassmorphic premium with teal/emerald primary accents
- Translucent surfaces, atmospheric blob backgrounds, film grain texture
- Rounded-everything, expressive, alive and personal

**Admin dashboard:**
- Clean ops tool — Vercel/Linear aesthetic. Calm, spacious, surgical
- Dark base with strong typographic hierarchy. Data readable at a glance
- Controls obviously clickable. No decoration for decoration's sake
- References: Vercel dashboard (tight, purposeful, elegant), Linear (whitespace-rich, zero clutter)
- Anti-references: Bootstrap admin panels, flashy rainbow dashboards, generic SaaS templates

**Theme**: Dark mode primary throughout. OKLCH color space. Primary: teal/emerald. Accent: pink/rose.

### Design Principles

1. **Personality over polish** — Every surface should feel like it belongs to MyGang. Character colors tint bubbles, wallpapers set mood, micro-labels use uppercase tracking.
2. **Glass and glow** — Translucent cards with backdrop-blur, atmospheric gradients, subtle shadows. The UI should feel like it floats.
3. **Touch-first, keyboard-friendly** — 44px minimum touch targets, pill-shaped buttons, safe area padding. WCAG AA non-negotiable.
4. **Motion with purpose** — Framer Motion for transitions. Always respect `prefers-reduced-motion`. Animation guides attention, never distracts.
5. **Signal over noise (admin)** — Surface only what requires attention or action. Destructive controls are visually distinct. Secondary data lives one click away.
6. **Calm authority (admin)** — Spacious layouts with confident typography. If something needs urgent attention, color and weight make it unmissable. Everything else is quiet.

### Design System Summary
- **Fonts**: Geist Sans (body), Geist Mono (code), Outfit (brand headings, 700-900 weight)
- **Radius**: Large — `rounded-3xl` sections, `rounded-full` buttons, `rounded-[22px]` chat input
- **Spacing**: Tailwind scale with `p-6` card padding, `gap-3` action items
- **Components**: shadcn/ui base with CVA variants, Radix primitives, Lucide icons
- **Animations**: Lazy-loaded Framer Motion (`m` components), CSS keyframes, Lottie for celebrations
- **Character colors**: Gold (Kael), Purple (Nyx), Steel Blue (Atlas), Pink (Luna), Orange-Red (Rico), Spring Green (Vee)
