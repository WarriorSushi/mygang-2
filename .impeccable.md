## Design Context

### Users
Broad casual audience seeking AI entertainment and emotional support. Users come for fun, companionship, and connection with AI characters that feel alive. The app should evoke comfort, delight, and a sense of belonging — like hanging out with friends who are always available.

### Brand Personality
**Bold, Playful, Premium.** The voice is casual and Gen-Z-coded ("gang," "vibe," "chaos gremlin") but the visual execution is polished and high-end. Think of it as a luxury clubhouse with a sense of humor.

### Aesthetic Direction
- **Visual tone**: Dark-first glassmorphic premium with teal/emerald primary accents, translucent surfaces, atmospheric blob backgrounds, film grain texture
- **References**: Character.ai (AI-first interaction patterns, structured chat) + Notion (clean minimal layouts, thoughtful whitespace)
- **Anti-references**: Generic SaaS dashboards, clinical/sterile AI interfaces, cluttered social media feeds
- **Theme**: Dark mode default with full light mode support. OKLCH color space for perceptual uniformity. Primary shifts from blue (light) to teal/emerald (dark). Accent shifts from lavender to pink/rose.

### Design Principles
1. **Personality over polish** — Every surface should feel like it belongs to MyGang, not a template. Character colors tint bubbles, wallpapers set mood, micro-labels use uppercase tracking.
2. **Glass and glow** — Translucent cards with backdrop-blur, atmospheric gradients, and subtle shadows create depth without heaviness. The UI should feel like it floats.
3. **Touch-first, keyboard-friendly** — 44px minimum touch targets, pill-shaped buttons, safe area padding. But keyboard navigation and screen reader support are non-negotiable (WCAG AA).
4. **Motion with purpose** — Framer Motion for transitions, CSS keyframes for ambient effects. Always respect `prefers-reduced-motion`. Animation should guide attention, not distract.
5. **Progressive disclosure** — Don't overwhelm. Show what matters, hide complexity behind intuitive interactions. Empty states and loading states are first-class citizens.

### Design System Summary
- **Fonts**: Geist Sans (body), Geist Mono (code), Outfit (brand headings, 700-900 weight)
- **Radius**: Large — `rounded-3xl` sections, `rounded-full` buttons, `rounded-[22px]` chat input
- **Spacing**: Tailwind scale with `p-6` card padding, `gap-3` action items, micro-spacing via fractional values
- **Components**: shadcn/ui base with CVA variants, Radix primitives, Lucide icons
- **Animations**: Lazy-loaded Framer Motion (`m` components), CSS keyframes for ambient effects, Lottie for celebrations
- **Character colors**: Gold (Kael), Purple (Nyx), Steel Blue (Atlas), Pink (Luna), Orange-Red (Rico), Spring Green (Vee)
